Selección de Herramientas de Desarrollo

A continuación se presentan las herramientas seleccionadas para el desarrollo de la aplicación One Language, orientada al reconocimiento de lengua de señas mediante inteligencia artificial.

Tipo de herramienta Herramienta seleccionada Función Justificación
IDE
Visual Studio Code
v1.x · Microsoft
 Entorno de desarrollo para programar, editar y ejecutar código. Es ligero, gratuito y soporta múltiples lenguajes (Python, JS). Tiene muchas extensiones útiles para IA y desarrollo web.
Lenguaje
Python
3.x · PSF
 Procesamiento de datos, IA y reconocimiento de señas. Es ideal para inteligencia artificial y visión por computadora (usa librerías como OpenCV, TensorFlow).
Frontend
React
18.x · Meta
 Desarrollo de la interfaz web interactiva. Permite crear interfaces dinámicas, rápidas y fáciles de mantener.
Backend
Django
4.x · DSF
 Manejo de lógica del sistema, API y comunicación con el frontend. Es seguro, robusto y funciona muy bien con Python (ideal para proyectos con IA).
Base de datos
PostgreSQL
16.x · PostgreSQL Global
 Almacenamiento de usuarios, historial de traducciones y datos del sistema. Es potente, escalable y muy usado en aplicaciones reales.
Control de versiones
GitHub
Git · Microsoft
 Gestión de versiones del código y trabajo colaborativo. Permite guardar cambios, trabajar en equipo y tener respaldo del proyecto.
Modelado
Draw.io
Diagrams.net
 Creación de diagramas (UML, base de datos, arquitectura). Es gratuito, fácil de usar y perfecto para documentar el sistema.
Librerías IA
OpenCV / MediaPipe
Google · Open Source
 Detección de manos y reconocimiento de señas. Permiten capturar movimientos en tiempo real desde la cámara.
Nota: Todas las herramientas seleccionadas son de código abierto o de uso gratuito, lo que garantiza accesibilidad para el equipo de aprendices del SENA y facilita su adopción en el entorno de formación.
