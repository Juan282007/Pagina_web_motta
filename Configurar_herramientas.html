Informe de Construcción del Software

Configuración de Herramientas de Desarrollo

1. Introducción

El presente informe describe las herramientas seleccionadas para el desarrollo del sistema de reconocimiento de señas, detallando sus características principales, así como los procedimientos de instalación y configuración necesarios para su correcto funcionamiento.

2. Características de las Herramientas

IDE: Visual Studio Code

  • Editor de código ligero y gratuito.
  • Compatible con múltiples lenguajes (Python, JavaScript, etc.).
  • Amplio soporte de extensiones.
  • Integración con Git.
  • Autocompletado inteligente (IntelliSense).

Lenguaje: Python

  • Lenguaje interpretado y fácil de aprender.
  • Amplio soporte para inteligencia artificial.
  • Librerías especializadas como:
    • OpenCV (visión por computadora).
    • TensorFlow (machine learning).
    • MediaPipe (detección de manos).

Frontend: React

  • Librería basada en componentes.
  • Permite interfaces dinámicas.
  • Renderizado rápido (Virtual DOM).
  • Reutilización de código.

Backend: Django

  • Framework robusto basado en Python.
  • Incluye sistema de autenticación.
  • Seguridad integrada.
  • Arquitectura MTV (Modelo - Template - Vista).

Base de Datos: PostgreSQL

  • Sistema relacional potente.
  • Soporte para grandes volúmenes de datos.
  • Alta seguridad y confiabilidad.
  • Compatible con Django.

Control de Versiones: GitHub

  • Almacenamiento de código en la nube.
  • Control de versiones.
  • Trabajo colaborativo.
  • Gestión de ramas (branching).

Modelado: Draw.io

  • Herramienta gratuita.
  • Creación de diagramas UML.
  • Interfaz sencilla.
  • Exportación en varios formatos.

Librerías IA: OpenCV / MediaPipe

  • Detección de imágenes en tiempo real.
  • Reconocimiento de manos y movimientos.
  • Integración con Python.
  • Ideal para visión por computadora.

3. Instalación y Configuración

3.1 Visual Studio Code

Instalación:

  • Descargar desde la página oficial.
  • Ejecutar instalador.
  • Seguir pasos por defecto.

Configuración:

  • Instalar extensiones:
    • Python
    • ES7+ React Snippets
    • Prettier (formateo)
    • Live Server
  • Activar auto guardado.
  • Activar formato automático.
  • Seleccionar tema (oscuro o claro).

3.2 Python

Instalación:

  • Descargar desde python.org.
  • Marcar la opción "Add Python to PATH".
  • Finalizar instalación.

Verificación:

python --version

Programa de prueba:

print("Hola Mundo")

3.3 React

Instalación:

  • Instalar Node.js.
  • Ejecutar:
npm create vite@latest
  • Seleccionar React.
  • Instalar dependencias:
npm install

Ejecución:

npm run dev

3.4 Django

Instalación:

pip install django

Crear proyecto:

django-admin startproject proyecto
cd proyecto
python manage.py runserver

3.5 PostgreSQL

Instalación:

  • Descargar desde la página oficial.
  • Configurar usuario y contraseña.

Crear base de datos:

CREATE DATABASE prueba;

Crear tabla:

CREATE TABLE usuarios (
    id SERIAL PRIMARY KEY,
    nombre VARCHAR(50)
);

3.6 GitHub

Configuración:

git init
git add .
git commit -m "Primer commit"
git remote add origin URL_DEL_REPO
git push -u origin main

3.7 Draw.io

  • Ingresar a la herramienta online.
  • Crear diagramas UML.
  • Guardar o exportar.

3.8 OpenCV y MediaPipe

Instalación:

pip install opencv-python mediapipe

Prueba básica:

import cv2
print("OpenCV funcionando")

5. Conclusión

Las herramientas seleccionadas permiten desarrollar un sistema completo de reconocimiento de señas, combinando tecnologías modernas de frontend, backend e inteligencia artificial. Además, su correcta instalación y configuración garantiza un entorno de desarrollo eficiente, escalable y listo para implementar funcionalidades avanzadas.
