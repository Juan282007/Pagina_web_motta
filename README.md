# PAGINA_WEB_MOTTA

## Paso a paso para ejecutar el proyecto en Visual Studio Code

Este proyecto está desarrollado en **HTML y CSS**, por lo tanto no necesita instalación de paquetes ni compilación.

---

# 1. Abrir Visual Studio Code

Abrir Visual Studio Code desde el escritorio o menú inicio.

---

# 2. Abrir la carpeta del proyecto

En VS Code:

- Clic en **File**
- Clic en **Open Folder**
- Buscar la carpeta:

```bash
PAGINA_WEB_MOTTA
```

- Seleccionar la carpeta
- Clic en **Open**

---

# 3. Verificar estructura del proyecto

Debe verse algo así:

```bash
PAGINA_WEB_MOTTA
│
├── CSS
│   └── style.css
│
├── Configurar_herramientas.html
├── Construccion_software.html
└── Herramientas_desarrollo.html
```

---

# 4. Abrir archivo principal

Buscar el archivo principal del proyecto:

```bash
Construccion_software.html
```

Dar doble clic para abrirlo.

---

# 5. Ejecutar el proyecto

Hay dos formas.

## Opción 1: Abrir directamente en navegador

- Clic derecho sobre:

```bash
Construccion_software.html
```

- Seleccionar:

```bash
Open with Live Server
```

Se abrirá automáticamente en el navegador.

---

## Opción 2: Abrir manualmente

- Buscar el archivo:

```bash
Construccion_software.html
```

- Doble clic.
- Se abrirá en Chrome, Edge o Firefox.

---

# 6. Verificar estilos CSS

Asegurarse que en el HTML esté conectado el CSS:

```html
<link rel="stylesheet" href="CSS/style.css">
```

Esto permite cargar los estilos del proyecto.

---

# 7. Modificar cambios

Si cambias HTML o CSS:

Guardar:

```bash
Ctrl + S
```

Si usas Live Server, se actualiza automáticamente.

---

# 8. Archivos del proyecto

## HTML
Contienen la estructura del sitio web.

- Configurar_herramientas.html
- Construccion_software.html
- Herramientas_desarrollo.html

## CSS
Contiene estilos visuales.

- CSS/style.css

---

# 9. Recomendaciones

Usar extensión de VS Code:

- Live Server
- HTML CSS Support
- Prettier
